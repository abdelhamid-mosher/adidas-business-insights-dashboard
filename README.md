# Adidas Business Insights Dashboard

An interactive Power BI dashboard designed to analyze Adidas's global business performance, tracking key metrics across sales, profit, and volume from 2023 to 2025. 

---

## Dashboard Overview
<img width="1287" height="442" alt="Adidas dashboard" src="https://github.com/user-attachments/assets/04c7ec36-6364-4419-9fd2-5ecf8b5730c2" />

The dashboard provides a macroscopic view of financial health alongside granular product and channel-level performance analytics. It is structured to help stakeholders identify growth drivers, seasonal trends, and optimization opportunities.

### Key Performance Indicators (KPIs)
* **Total Sales:** \$287,377
* **Total Profit:** \$85,069 (~29.6% profit margin)
* **Units Sold:** 3,550 PCS

### Core Visualizations & Insights
* **Sales & Quantity Distribution:** Breakdown of sales by product categories (**Footwear** dominates at 62%, followed by Apparel and Accessories) and quantity percentage by store type (**Online** and **Retail** channels drive the highest volumes).
* **Temporal Trends:** A quarterly sales timeline and monthly profit trend charts spanning 2023 to 2025 to monitor seasonality and cyclical performance.
* **Product Profitability:** A ranked bar chart highlighting top-performing products, identifying the **Predator Freak** and **Ultraboost Light** as primary profit drivers.
* **Geographic Mapping:** Spatial analysis of regional performance concentrated across the Middle East and North Africa (MENA) region.

---

## Features
* **Dynamic Filtering:** Interactive slicers to filter data by specific `Product_Name`, `Category`, and `Order_Date` (Years: 2023, 2024, 2025).
* **Cross-Filtering:** Selecting any visual automatically updates the rest of the dashboard to show contextual insights.
* **High Scannability:** Clean UI utilizing Adidas's brand identity with distinct sections for quick decision-making.

---

## Technical Stack
* **Tool:** Power BI Desktop
* **Data Source:** Excel / SQL Database (Adjust as applicable)
* **Languages:** DAX (Data Analysis Expressions) for custom metrics and Power Query M for data transformation.

---

## How to Use the Dashboard
1. Download the `.pbix` file from this repository.
2. Open the file using [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
3. Use the left-hand panel slicers to filter data by product name, category, or specific timelines.
